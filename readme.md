1. The Squad (Hardcoded Data)

Players: Maor, Alko, Becker, Regev.

2. Session Logic

Creation: A host selects a Game and checks off which Players participated.

State: A session is created in the DB and appears in the "Active Sessions" list on the homepage.

Participation: Only players marked as participants in the creation step appear in the voting list.

3. Voting System

Flow: Users click "Join" on a session -> Select their identity ("Who are you?") -> Drag and drop players to rank them.
